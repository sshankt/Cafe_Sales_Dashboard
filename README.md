# ☕ Cafe Sales Data Analysis Project
### 📌 Project Overview

This project focuses on analyzing a real-world cafe sales dataset to extract meaningful insights and build an interactive dashboard. The goal was to handle real-world data challenges like missing values and present the data in a clear and structured way.

## 🎯 Objectives

Understand and explore the dataset
Clean and preprocess raw data
Handle missing values effectively
Create an interactive dashboard
Present insights in a simple and understandable way

## 📂 Dataset

Source: Kaggle

Type: Cafe Sales Data
Contains information like:
Item names
Quantity sold
Revenue
Payment methods
Location (In-store / Takeaway)
Order date

## 🛠️ Tools & Technologies Used

Python (Pandas) → Data cleaning & preprocessing
Jupyter Notebook → Data analysis
Microsoft Excel → Dashboard creation
Charts & Pivot Tables → Data visualization

## 🧹 Data Cleaning Process

Loaded dataset into Jupyter Notebook
Checked structure and data types
Identified null/missing values in multiple columns
Handling Missing Values:
Used mean, median, and mode for appropriate columns
Found ~30% missing data in two columns

## 📊 Dashboard (Excel)

An interactive dashboard was built to present structured insights:

### Key Components:

KPI Metrics: Total Revenue, Total Orders, Average Order Value
Sales Analysis: Item-wise revenue and quantity
Time Analysis: Monthly trends
Operational View: Sales by location (In-store vs Takeaway)
Customer Behavior: Payment method distribution
Data Quality: Missing values visualization
